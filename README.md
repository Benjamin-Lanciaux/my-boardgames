# My Boardgames

Application web personnelle de gestion de collection de jeux de societe, inspiree de BoardGameGeek.

## But

Suivre ma collection de jeux de societe importee depuis BGG, avec:
- des categories et tags personnalisables
- des notes personnelles
- la derniere partie jouee

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS v3
- shadcn/ui
- React Router v6
